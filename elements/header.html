<header>

English Base Word: Header
Spanish Word Equivalent: Encabezamiento

Definición

El <header> es una etiqueta de elemento de bloque que se crea en la parte superior de la página HTML. También contiene información de introducción como el título, logo o información acerca del autor. El <header> es un elemento sistemático que es utilizado en reemplazo del <div> para ayudar a encontrar los mecanismos de información relevante a su página.

Definition

The <header> tag is a block element that creates the header at the top of a HTML page. It also contains introductory content, such as a title, logo, or information about the author. The header is a semantic element used in place of a <div> tag to help search engines find relevant information about your page.

¿Cómo recordar?

Secciones de una página de HTML puede ser similar a las secciones de un documento escrito. La parte superior de un documento es conocido como el header, al igual que la parte suLasperior de una página HTML es conocido como el <header>.

How to Remember

The main sections of an HTML page are the same as the sections of a written document. The top part of a document is known as the header, just like the top part of an HTML page is known as the <header>.

<header>
  <h1>Title of the content</h1>
  <div class="example">
    <p>This is a paragraph example.</p>
  </div>
<header>
